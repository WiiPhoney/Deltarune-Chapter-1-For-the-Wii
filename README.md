# Deltarune Chapter 1 for the Wii

This is an early work-in-progress port of *Deltarune Chapter 1* to the Nintendo Wii.

The project is in a very incomplete state. Only a single room is currently implemented, and bugs, missing features, and instability are expected. This repository is being shared to document progress and gather early feedback.

## Current Features

* 2 player support (planned to expand to 4 players)
* Orientation-based controls (controls adapt based on how the Wii Remote is held)
* Textbox sounds and select audio effects played through the Wii Remote speaker (work in progress)

## Work in Progress

* Dialogue and interactions
* Additional rooms
* Full 4 player support
* Expanded and improved sound effects

## Notes

* Expect crashes, bugs, and incomplete functionality
* This is a hobby project and development may be slow

Feedback, bug reports, and suggestions are welcome. If you test the project, please report any issues you encounter.
